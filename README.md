# DevOps Infrastructure Provisioning Tool

## Overview
A Python-based tool that simulates infrastructure provisioning and service configuration.

## Project Structure
```
infra-automation/
├── scripts/
├── configs/
├── logs/
├── src/
└── README.md
```

## Setup
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Run
```bash
cd src
python infra_simulator.py
